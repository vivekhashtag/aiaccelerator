## The Seven Fundamental Gates

Every digital circuit — from a simple timer to a 100-billion-parameter LLM inference chip — is built from combinations of these seven fundamental logic gates.

The gates you will learn: **NOT, AND, OR, NAND, NOR, XOR, XNOR**.

NAND is the most important for hardware: any Boolean function can be implemented using only NAND gates. A chip foundry only needs to characterise, optimise, and manufacture one type of gate to build any digital circuit.
